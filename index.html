Understanding Recursion

Fibonacci Sequence

The Fibonacci sequence is defined as each number being the sum of the two preceding ones: F(n) = F(n-1) + F(n-2).

Iterative approach: Uses a loop to build the sequence efficiently with O(n) time complexity.

Recursive approach: Directly implements the mathematical definition but has exponential time complexity O(2^n) without memoization.

The recursive version demonstrates how a problem can be broken down into smaller subproblems of the same type.

Merge Sort

Merge sort is a classic divide-and-conquer algorithm that recursively splits an array into halves until single elements remain.

Base case: An array with 0 or 1 element is already sorted.

Recursive case: Split the array, recursively sort both halves, then merge them back together in sorted order.

Merge sort has O(n log n) time complexity, making it much more efficient than simple sorting algorithms like bubble sort for large datasets.
